Código criado para a disciplina de Análise de Algoritmos do Mestrado em computaçao Aplicada da UNIVALI 2026/01.
Implementar a solução do caixeiro viajante utilizando uma das heurísticas apresentadas em aula.

A heurística utilizada: vizinho mais próximo.
O código faz a leitura de uma tabela de distancias e executa o algoritmo mostrando o trajeto, a distancia entre as cidades percorridas e o valor total do percurso.
